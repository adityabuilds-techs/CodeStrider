Python Functions

📖 What are Functions?

Reusable Code

A function is a reusable block of code that performs a specific task. Instead of writing the same code many times, you write it once and call it whenever needed.

💻 Syntax

def greet():
    print("Hello!")

greet()

📥 Functions with Parameters

def greet(name):
    print("Hello", name)

greet("Aditya")

📤 Returning Values

def add(a,b):
    return a+b

result = add(10,20)

print(result)

Python Loops

📖 What are Loops?

Repeat Tasks

Loops execute the same block of code multiple times without writing it again.

💻 For Loop

for i in range(5):
    print(i)

💻 While Loop

count = 1

while count <= 5:
    print(count)
    count += 1

📝 Practice

Print numbers from 1 to 10.

Print your name 5 times.

Print all even numbers from 2 to 20.

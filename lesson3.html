If Statements

📖 What are If Statements?

Decision Making

Programs often need to make decisions. The if statement allows Python to execute code only when a condition is true.

💻 Syntax

age = 18

if age >= 18:
    print("You can vote")

🧪 Example

marks = 80

if marks >= 35:
    print("Pass")

📝 Practice

Create programs that check:

  • Age greater than 18
  • Marks greater than 35
  • Number is positive

🎯 Mini Challenge

Ask the user for their age. If age is 18 or above, print: 

Eligible to Vote

Otherwise print: 

Not Eligible

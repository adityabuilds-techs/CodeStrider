Python Variables

📖 What is a Variable?

Variables

A variable is a container used to store data in Python. Think of it as a labelled box where you keep information that your program can use later.

💻 Syntax

name = "Aditya"

age = 16

print(name)

print(age)

📌 Variable Naming Rules

✅ Variable names can contain letters, numbers and underscores.

✅ They cannot start with a number.

✅ They cannot contain spaces.

✅ Python keywords cannot be used.

📚 Data Types

String

name = "Aditya"

Integer

age = 16

Float

height = 5.9

Boolean

isStudent = True

Python Input & Output

📖 What is Input & Output?

Input

Input allows the user to enter data into a program. Python uses the input() function to receive information.

Output

Output displays information on the screen. Python uses the print() function to show results.

💻 Syntax

name = input("Enter your name: ")

print(name)

🧪 Example

name = input("Enter your name: ")
age = input("Enter your age: ")

print("Hello", name)
print("You are", age, "years old.")
